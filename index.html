About Me

Autonomous Drone integrating AI for Object Detection

Developed as a final year project, this system is a computer vision-guided autonomous tracking system for drones. It utilizes a ground run YOLOv5 model for real-time person detection from a live video feed. A Python control loop processes this data, calculating proportional control outputs for yaw, pitch, and throttle. These commands are sent as MAVLink RC overrides via DroneKit, enabling the drone to actively track its target. The core logic operates as a finite state machine, managing states for target acquisition, tracking, and occlusion handling.

Manufacturing & Supplier Analysis

Developed during my internship at Schneider Electric, this Python script provides manufacturing cost analysis and supplier ranking. The project uses Pandas and Openpyxl to read production data from an Excel file, calculate the financial impact of part weight discrepancies, and rank suppliers based on these metrics. The script automates the entire process, appending the final analysis—including per-part cost differences and total monthly loss—back into the original spreadsheet for clear reporting and decision-making.
